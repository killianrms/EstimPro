Politique de confidentialité

Dernière mise à jour : Décembre 2024

1. Qui sommes-nous ?

EstimPro est un service d'estimation immobilière en ligne. Nous nous engageons à protéger et respecter votre vie privée.

2. Quelles données collectons-nous ?

Nous collectons les informations suivantes lorsque vous utilisez notre service d'estimation :

  • Données personnelles : nom, prénom, adresse email, numéro de téléphone
  • Données sur le bien : adresse, type de bien, superficie, nombre de pièces, état, etc.
  • Données techniques : adresse IP, type de navigateur, pages visitées
  • Données de projet : raison de l'estimation, délai du projet

3. Pourquoi collectons-nous ces données ?

Nous utilisons vos données pour :

  • Calculer une estimation de votre bien immobilier
  • Vous contacter pour affiner l'estimation
  • Améliorer nos services d'estimation
  • Vous proposer des services complémentaires (avec votre consentement)
  • Respecter nos obligations légales

4. Base légale du traitement

Le traitement de vos données repose sur :

  • Votre consentement pour l'estimation et les communications commerciales
  • L'exécution du service que vous avez demandé
  • Notre intérêt légitime à améliorer nos services

5. Avec qui partageons-nous vos données ?

Vos données peuvent être partagées avec :

  • Nos partenaires : agents immobiliers, diagnostiqueurs, experts
  • Prestataires techniques : hébergement, maintenance, analytics
  • Autorités : uniquement si requis par la loi

Nous ne vendons jamais vos données à des tiers.

6. Combien de temps conservons-nous vos données ?

  • Données d'estimation : 3 ans maximum
  • Données de contact : jusqu'à votre demande de suppression
  • Données techniques : 13 mois maximum

7. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : connaître les données que nous avons sur vous
  • Droit de rectification : corriger vos données
  • Droit d'effacement : demander la suppression de vos données
  • Droit d'opposition : vous opposer au traitement
  • Droit à la portabilité : récupérer vos données
  • Droit de limitation : limiter le traitement

8. Comment exercer vos droits ?

Pour exercer vos droits, contactez-nous :

  • Email : privacy@estimpro.com
  • Courrier : EstimPro - Service Données Personnelles - [Adresse]

Nous vous répondrons dans un délai maximum de 30 jours.

9. Sécurité des données

Nous mettons en place des mesures techniques et organisationnelles appropriées pour protéger vos données :

  • Chiffrement des données sensibles
  • Accès limité aux données
  • Surveillance des systèmes
  • Formation du personnel

10. Cookies

Nous utilisons des cookies pour améliorer votre expérience. Vous pouvez gérer vos préférences cookies dans les paramètres de votre navigateur.

11. Modifications

Cette politique peut être modifiée. Les modifications importantes vous seront notifiées par email ou sur notre site.

12. Contact

Pour toute question sur cette politique :

  • Email : contact@estimpro.com
  • DPO : dpo@estimpro.com

13. Réclamation

Vous avez le droit de déposer une réclamation auprès de la CNIL : www.cnil.fr
